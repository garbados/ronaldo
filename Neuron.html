Interface: Neuron

Neuron

A generic interface for Neurons. It includes getters and setters for a neuron's weight and biases. Classes inheriting from this interface should implement a `_process` function that takes an array of bits of size equal to the neuron's list of weights.

process(input)

Given an array of input bits, return the neuron's output. If the array of input bits is a different length than the neuron's list of weights, it will throw an error. If `_process` is unimplemented, it will also throw an error.
